# 🧮 Calculator Web App

A modern, responsive, and user-friendly calculator built using **HTML, CSS, and JavaScript**.
Designed with a clean UI and smooth user experience, including full keyboard support.

---

## 🚀 Live Demo

[👉 *(Add your GitHub Pages link here after deployment)*](https://amanmohanty18.github.io/calculator-web-app)

---

## 📌 Features

* ➕ Basic arithmetic operations (Addition, Subtraction, Multiplication, Division)
* 🎨 Modern UI with glassmorphism design
* ⌨️ Full keyboard support
* 📱 Fully responsive (mobile-friendly)
* ⚡ Fast and lightweight
* ❌ Error handling (e.g., division by zero)

---

## 🛠️ Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling & responsive design
* **JavaScript (Vanilla JS)** – Functionality
